Juego del ahorcado

Este proyecto se realizo para el challeger 2 del ciclo fundamentos de programación WEB del curso OracleOne Next Education, fue desarrollado con JAVASCRIPT, CSS y HTML.

El desarrollo del dibujo del ahorcado se realizo sobre un canvas, el juego tiene un listado de palabras fijas para el juego, sin embargo es posible agregar nuevas palabras segun el gusto o la necesidad del usuario, ademas cuenta con un diseño responsivo para PC y celulares. El juego solo recibe letras mayusculas para la interaccion con el ahorcado.
